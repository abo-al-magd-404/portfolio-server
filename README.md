# Portfolio Server

Welcome to the Portfolio Server! This server is designed to serve information about various portfolio projects. You can find details about my work, skills, and personal projects.

## Features
- List of projects
- Skills overview
- Contact information

## Getting Started
To get started with the server, clone the repository and run:

```bash
yarn install
yarn start
```
